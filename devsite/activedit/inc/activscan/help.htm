Specifies filename on server. Overrides users capability of naming file.

onsave attribute:

The onsave attribute can be one of the following options:

  • alert: Display s an alert window with the message contained in the onsaveaction attribute.
  • redirect: Redirects the user to the URL contained in the onsaveaction attribute.
  • call: Calls the javascript command contained in the onsaveaction attribute.
         This is in the form of  "foobar( );"
Sample Implementation
<cf_activescan 
  inc="http://me.com/admin/inc/"
  imagepath="c:\uploads\"
  license="abcdefgh"
  onsave="redirect"
  onsaveaction="http://me.com/success.cfm?file=%file%"
>

Tips:

  • Clients need to use Internet Explorer 4 or above (tested on IE 4, IE 5)
  • You will need one 'inc' directory per domain name.
  • In the onsaveaction attribute, you can insert the filename saved by including the identifier "%file%" within the string.

    Examples:
         onsave="alert"
         onsaveaction="Your file was saved as %file%"
        
         onsave="call"
         onsaveaction="doSomething( '%file%' )"

         onsave="redirect"
         onsaveaction="http://me.com/success.cfm?filename=%file%"
